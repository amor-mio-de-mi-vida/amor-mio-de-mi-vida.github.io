Three ingredients of a machine learning algorithm

Every machine leaning algorithm consists of three different elements - The hypothesis class: the "program structure", parameterized via a set of parameters, that describes how we map inputs to outputs. - The loss function: a function that specifies how "well" a given hypothesis performs on the task of interest. - An optimization method: a procedure for determining a set of parameters that (approximately) minimize the sum of losses over the training set. ## Softmax regression ### Linear hypothesis function ### Loss function simple version: this loss function is very bad for optimization (selecting the best parameters).
